# Hospital Appointment Management System

## Overview
A **full-stack hospital appointment system** built with **React, Node.js, Express.js, and MySQL**.  
Supports **role-based dashboards** for **patients, doctors, and admin**.  
Allows patients to book appointments, doctors to approve/reject, and admin to view all users and appointments.

------->

## Features

### Patient
- Register & login
- Book appointments with doctors
- View appointment status (pending, approved, rejected)

### Doctor
- Login
- View all patient appointments
- Approve or reject appointment requests

### Admin
- Login
- View all users (patients and doctors)
- View all appointments

-------------------------------

## Technologies I have Used
- **Frontend:** React, Axios  
- **Backend:** Node.js, Express.js  
- **Database:** MySQL  
- **Authentication:** JWT  
- **API Requests:** RESTful APIs

-------------------------------

## Installation

### Backend
```bash
cd backend
npm install
npm run dev
```
### Frontend
```bash
cd frontend
npm install
npm start
```
